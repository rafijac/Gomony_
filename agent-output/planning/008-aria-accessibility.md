## Value Statement
As a visually impaired or keyboard-only user, I want Gomony’s board, cells, and controls to have explicit ARIA roles, robust keyboard navigation, and responsive design, so that I can play and interact with the game confidently across devices and assistive technologies.

## Objective

- Keyboard navigation and focus management
- Mobile responsiveness using @media queries

- Edge case handling and test coverage for all user roles
