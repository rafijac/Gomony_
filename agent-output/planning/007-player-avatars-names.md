# Gomony Player Avatars & Names Plan (Final)

## Value Statement
As a player, I want to see player avatars and names in the Gomony UI, so that I can easily identify myself and my opponent, improving immersion and social connection.

## Objective
Enable display, selection, and update of player avatars and names throughout the game UI, with robust backend support, migration for legacy users, and real-time update propagation for active sessions.

## Requirements
- Migration/fallback logic for legacy users
- Test cases for all roles and edge cases
- Preset avatar set and fallback/default behavior
- Update propagation for active sessions (backend sync, locked after game start)
- Avatars/names editable only in lobby, locked after game start
- All open questions resolved as of 2026-03-25 (see Expert answers)
